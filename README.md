# Weather App

Check the current weather In Nantes city. Switch between metric and imperial units.

## Features

1- Current local time and date

2- Temperature

3- Wind speed and direction

4- Precipitation

5- Metric vs Imperial system

6- Automatic refresh every hour

7- Error handling and loading info

## Installation

git clone https://github.com/thomasguesneau-data/open-meteo.git

API Open-Meteo

`npm install`

`npm run dev`
